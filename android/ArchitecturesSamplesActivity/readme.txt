- A sample project demonstrating 3 architectures
    - model view controller
    - model view presenter
    - model view view-model

- One main activity with 3 buttons to show implementation for each architecture
- Each implementation uses 2 logics
    - a button for incrementing a counter an displaying it on the screen
    - a button for inputting a text and displaying it on the screen

- android libraries used:
    - android live data, view-model
